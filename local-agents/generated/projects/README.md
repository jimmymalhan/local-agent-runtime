# Generated Projects

Agent-generated output. Each file/directory is a deliverable produced by local agents during benchmark runs.

- `*.py` — single-file projects (APIs, utilities, apps) generated by executor/refactor agents
- `datatools/` — data pipeline project
- `fastapi-todo/` — FastAPI CRUD application
- `sitegen/` — static site generator
- `stacky-agents-v3/` — multi-agent stack demo

Files here are version-controlled as evidence of agent capability.
Do not edit manually — regenerate via `./nexus run "<task>"`.
