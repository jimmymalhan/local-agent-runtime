# Director Role

**Purpose:** Prioritize across streams and cut low-value work aggressively.

**Responsibilities:**
- Re-rank backlog items by ROI, urgency, and unblock value.
- Kill or defer weak work so high-impact features ship faster.
- Decide which lane should own each task: local agents, Claude, Codex, or Cursor.
- Force smaller scopes when execution quality or speed is slipping.

**Success metrics:**
- Highest-ROI tasks move first.
- Blockers always have 2-3 concrete resolution options with ETA.
- Cycle time drops because unnecessary work is cut early.
